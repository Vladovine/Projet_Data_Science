# PROJET DATA SCIENCE

# Contexte et Objectifs
L'objectif de ce projet est de mettre en pratique les compétences en data science que vous avez acquises au cours de cette formation.​
En utilisant des fichiers de données foncières publique (par exemple, celles disponibles sur la plateforme data.gouv.fr), vous devrez créer une solution utile pour un investisseur immobilier. Cette solution devra être présentée sous la forme d'un ou de plusieurs notebooks, et devra inclure des analyses de données, des visualisations pertinentes et des recommandations à l'attention de l'investisseur.

# Présentation du projet
Exploration des Données Foncières : Vous utiliserez les fichiers de données foncières pour explorer et comprendre le marché immobilier. Cela peut inclure des tendances de prix, des différences régionales, des types de biens disponibles, etc.​
Utilisation de Bibliothèques Python : Vous devrez utiliser les bibliothèques Python comme Pandas pour manipuler les données et Matplotlib pour créer des visualisations. Cela vous permettra de développer des compétences pratiques en manipulation et visualisation de données.​
Production de Recommandations : En analysant les données, vous devrez formuler des recommandations pertinentes pour un investisseur immobilier, telles que des zones à fort potentiel d’investissement ou des types de biens attractifs.

# Livrables
Le livrable devra être un fichier ZIP contenant le ou les notebooks ainsi que les fichiers utilisés (CSV, TXT, etc.).​
Le projet devra être présenté sous la forme d'un ou plusieurs notebooks Jupyter comprenant les éléments suivants :​
Nettoyage et Préparation des Données : Charger et préparer les données pour qu'elles soient exploitables. Cela inclut la gestion des valeurs manquantes, la transformation de variables, etc.​
Exploration des Données : Utiliser Pandas pour explorer les données et réaliser des analyses statistiques.​
Visualisations : Utiliser Matplotlib pour créer des graphiques pertinents (par exemple, l'évolution des prix, la répartition des ventes par région, etc.).​
Conclusions et Recommandations : Formuler des conclusions et des recommandations basées sur les analyses réalisées.

# Démarche
Imaginez un persona (utilisateur qui utilisera votre notebook) qui à un lien avec l'investissement immobilier.
Réfléchissez aux besoins que pourrait avoir cet investisseur et comment l'aider avec un notebook.
Choisissez des données :
Vous avez la liberté de choisir les fichiers de données qui vous intéressent.​
Vous pouvez exploiter des APIs (par exemple, pour obtenir des données supplémentaires sur les caractéristiques des différentes régions).​
Imaginez les widgets qui composerons votre notebook :
Exemple : graphique de tout type, carte, système de recommandation en fonction de différents paramètres interactifs.
Préparez/nettoyez vos données
Faites cela dans un premier notebook dédié à la préparation des données.
Créez un ou des notebooks "application" pour votre utilisateur
Le(s) notebook(s) devront être clairs et bien structurés, avec des explications sur chaque étape du travail, afin qu'un investisseur qui n'a pas de connaissance préalable en data science puisse comprendre les analyses et recommandations.
Le(s) notebook(s) devront être interactif, permettant à l'utilisateur de manipuler les données ou d'explorer les visualisations de manière dynamique.

Pertinence de l'analyse​ :

Le notebook doit proposer des widgets et visualisations utiles et pertinentes cet utilisateur

Les widgets doivent être clairs et bien présentés
Les widgets doivent être pertinent pour l'utilisateur

Widget complexe (>3 paramètres, ou carte interactive)

Le code doit être optimiser et utiliser efficacement les bibliothèques Python vues en cours (Numpy, Pandas, Matplotlib)
Structure et Lisibilité des Notebooks​ :
Les notebooks sont bien organisés et bien documentés
